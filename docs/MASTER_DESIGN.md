## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **게임명** | Anima (아니마) |
| **장르** | 브라우저 기반 방치형 RPG (Idle RPG) |
| **플랫폼** | 웹사이트 |
| **벤치마크** | Melvor Idle |
| **수익 모델** | 기본 무료 + 유료 콘텐츠 확장팩 방식 |
| **한 줄 소개** | 자신만의 주력(呪力)을 창조하며 성장하는 방치형 RPG |

### 핵심 차별점 (vs Melvor Idle)
- **주력 커스터마이징**: 개념을 수집하고 조합하여 완전히 자신만의 능력을 설계
- **풍부한 내러티브**: Melvor Idle의 약점인 스토리 부재를 보완, 단 암시적·간접적 방식으로 전달
- **에테르 세분화 시스템**: 단순 마나량이 아닌 기맥·단전 등 체계적인 에너지 육성 구조

## 2. 세계관

### 2-1. 시대 배경
게임의 배경은 현대로부터 먼 미래의 **포스트 아포칼립스** 세계다.  
인류는 뇌 과학의 발전을 통해 생체 에너지 **에테르(Ether)** 의 존재를 발견하고, 이를 제어·증폭하는 기술을 개발하여 전례 없는 문명적 진보를 이루었다.

### 2-2. 에테르란
에테르는 모든 생명체 내부에 흐르는 생체 에너지다. 판타지 장르의 마나, 기(氣), 차크라에 해당하는 개념이지만, 아니마 세계관에서는 과학적으로 규명된 실재하는 물질이다.

- 에테르에 의도적으로 **프로세스를 기록**함으로써 힘의 크기·방향·속성 등을 설정하고 외부에 작용시킬 수 있다
- 개개인이 보유한 에테르 총량은 제한적이며, 소모된 에테르는 생명 활동으로 자연 회복된다

### 2-3. 문명의 흥망
에테르 기술은 급격히 상업화되었고, 사익을 위한 무분별한 연구(인체 실험, 유전자 조작, 강제 에테르 착취 등)가 만연했다.  
사회 제도가 기술 발전을 따라가지 못하면서 윤리 붕괴 → 사회 갈등 심화 → 테러·전쟁 → 환경 오염 → **문명 붕괴**의 수순을 밟았다.

### 2-4. 현재 (게임 시점)
- 오랜 세월이 흘러 자연이 번창했고 인류 문명은 자연속에 풍파되었다
- 유전자 조작과 환경 오염 그리고 방대한 에테르등의 영향으로 기괴하게 변형된 괴수의 생태계가 형성되었다
- 적대적인 AI 기계 생명체들이 존재한다
- 인간을 비롯한 생존 개체중 몇몇은 오랜 시간동안 축적된 대기중의 에테르를 이용하여 주력을 사용한다
- 소수의 인류 세력이 전 세계 각지에 흩어져 각자의 소규모 사회를 이루고 있다
- 살아남은 인류는 괴수와 기계 생명체들 사이에서 생존해 나가고 있다

### 2-5. 세력과 문화권
생존한 세력들은 각자의 역사적·문화적 뿌리를 바탕으로 독자적인 문화권을 형성했다.  
이를 통해 다양한 판타지 장르의 요소를 자연스럽게 공존시킨다.

| 세력 콘셉 | 특징 |
|-----------|------|
| 일본 전국시대풍 | 무녀, 주술, 민속 신앙 |
| 중세 왕국풍 | 기사, 마법, 중앙집권제 |
| 무협풍 | 무공, 내공, 문파 |
| 구 문명 계승 집단 | 고도의 기술력, 과거 문명의 흔적 보존 |

*(세력 상세 내용은 `/docs/story/world_lore.md` 참조)*

## 3. 스토리 & 주인공

### 3-1. 주인공 — 아니마(Anima)
주인공은 외형과 사고방식이 인간과 거의 동일하지만, 실은 **고도의 기술이 집약된 안드로이드**다.  
문명 붕괴 직전, 어느 비밀 집단이 "인간의 보존"을 목적으로 제작한 유일무이한 존재다.

- 인류 말기의 인간은 유전자 조작과 사이버네틱 개조로 이미 "인간"이라 부르기 어려운 상태였다
- 주인공은 그 어떤 사상·역사·정보도 사전 주입받지 않고, 갓 태어난 아기처럼 순수한 상태로 설계됐다
- 생체 기계이므로 에테르 축적 효율이 일반 생명체보다 월등히 높다
- 스스로는 인간이라 믿고 깨어나며, 점차 자신의 정체를 깨닫는다

### 3-2. 시작
주인공은 어느 폐허 속에서 **기억을 전부 잃은 채** 깨어난다.  
생존 본능만을 가지고 이 위험한 세계에 적응하고, 자신의 정체와 세계의 숨겨진 진실을 서서히 알아간다.

### 3-3. 스토리 방향성
- 고정된 메인 스토리 없이, **플레이어의 선택과 행동**이 각자의 스토리를 만든다
- 이것이 "아니마(Anima = 영혼)"라는 게임 타이틀의 의미: 플레이어 각자가 자신만의 영혼을 빚는다

### 3-4. 스토리텔링 방식
방치형 RPG의 특성상 스토리보다 육성이 주된 재미이므로, 스토리는 **암시적·간접적**으로 전달한다.

- 강제 시청 컷씬이나 캠페인 없음
- 게임 세계 곳곳에 **사소한 단서, 오래된 문서, 유물의 설명문** 등을 배치
- 플레이어가 주도적으로 탐구할 때만 스토리가 드러나는 구조
- 모든 스토리를 수집하지 않아도 게임 플레이에 지장 없음

## 4. 핵심 게임 시스템

### 4-1. 활동 시스템
Melvor Idle의 Skill에 해당하는 개념. 플레이어가 활동을 선택하면 캐릭터가 자동으로 반복 수행한다.

**활동 목록 (초안)**
| 활동 | 설명 |
|------|------|
| 탐사 | 지역을 탐색하며 개체와 전투, 유물·단서 발견 |
| 수련 | 에테르 총량 및 기맥·단전 강화 |
| 제작 | 장비, 소모품, 주력 재료 제작 |

*(활동 상세 설계는 `/docs/systems/skills.md` 참조)*

### 4-2. 계획 시스템
반복 수행이 아닌, **플레이어가 직접 판단하고 결정하는 행위**들.

| 계획 | 설명 |
|------|------|
| 조율 | 수집한 개념을 조합하여 자신만의 주력을 창조 |
| 세력 교류 | 각 세력과의 관계를 구축하고 협력·대립 결정 | 

### 4-3. 개념 시스템
게임의 핵심 수집 요소. 세계의 모든 사상 및 현상을 **사전적 의미의 개념 단위**로 수집·도감화한다.

**개념의 계층 구조**
- 개념을 조합하여 **주력**을 만들 수 있다
- 만들어진 주력은 다시 상위 주력의 **재료 개념**이 된다
- 단, 무한한 분해를 방지하기 위해 **최소 단계(원초 개념)** 가 존재한다

**예시 — 파이어볼 주력 설계 트리**
```
파이어볼
├── 불 (주력)
│   ├── [원초] 위력 (힘의 크기)
│   └── [원초] 진동
├── [원초] 구체 (형태)
├── [원초] 방향 (힘의 방향)
└── [원초] 위력 (힘의 크기)
```

**개념의 주요 수집처**
- 전투: 적 개체의 행동·주력을 관찰하여 개념 학습
- 탐사: 유물, 문헌, 환경 관찰
- 수련: 내면의 에테르 흐름 탐구

*(개념 시스템 상세는 `/docs/systems/skills.md` 참조)*

### 4-4. 주력(呪力) 시스템
이 게임의 가장 큰 차별점. 개념을 조합하여 자신만의 능력을 설계한다.

- 중세 마법, 주술, 신성, 내공, 무공 등 **모든 이능력의 범주를 포괄**하는 상위 개념
- 단순 파이어볼 같은 기성 마법이 아닌, 속성·형태·크기·방향 등을 자유롭게 조합
- 주력이 완성되면 전투 활동에서 자동으로 사용 가능

*(주력 시스템 상세는 `/docs/systems/combat.md` 참조)*

### 4-5. 에테르 시스템
단순한 마나량 개념을 넘어, **체내 에너지 구조를 세분화한 육성 시스템**.

| 요소 | 설명 |
|------|------|
| 에테르 총량 | 사용 가능한 에너지의 전체 용량 |
| 기맥(氣脈) | 체내 에테르의 흐름 경로. 소주천·대주천 구조로 확장 |
| 단전(丹田) | 에테르를 압축·저장하는 핵심 기관. 형성 및 강화 가능 |

- 수련 활동을 통해 기맥을 확장하고 단전을 형성
- 장비·유물·레벨업으로도 에테르 관련 수치 강화 가능
- 주인공(안드로이드)은 일반 생명체 대비 월등한 에테르 축적 효율 보유

*(에테르 시스템 상세는 `/docs/systems/progression.md` 참조)*

### 4-6. 콘텐츠 해금 구조
모든 콘텐츠가 처음부터 노출되지 않는다. **스토리 진행에 따라 단계적으로 활성화**된다.

- 예: 마을 재건 콘텐츠는 관련 스토리 트리거 전까지 UI에 노출되지 않음
- 이를 통해 초반 복잡성을 줄이고 자연스러운 성장감을 제공
